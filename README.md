# 🚀 Estelar

Jogo 2D desenvolvido como projeto pessoal utilizando a Godot Engine é um jogo que teve um início durante as aulas de Desenvolvimento de Jogos no IFSMG, com foco na aplicação prática de conceitos de programação e desenvolvimento de jogos.

## 🎮 Sobre o projeto

**Estelar** é um jogo criado com o objetivo de consolidar conhecimentos em lógica de programação e estruturação de sistemas interativos.

O projeto foi desenvolvido de forma autoral após estudos iniciais em desenvolvimento de jogos no IFSMG e sendo aprimorado durante essa minha nova jornada, servindo como evolução prática do aprendizado.

## 🧠 Conceitos aplicados

- Lógica de programação
- Controle de estados (game states)
- Sistema de eventos
- Interação com o jogador
- Organização de código
- Estruturação de cenas (Godot)

## 🛠 Tecnologias

- Godot Engine  
- GDScript  

## 🚀 Objetivo

Este projeto tem caráter educacional e representa minha evolução no desenvolvimento de jogos e na aplicação prática de conceitos fundamentais de programação.
